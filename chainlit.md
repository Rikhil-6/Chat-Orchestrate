# Chat Orchestrate

Use this space to coordinate multiple AI agents around a named project workspace.

Try:

```text
/spaces
/use default
Give me a kickoff plan for this project.
```

The local preview mode uses deterministic agent responses until `USE_OPEN_SWARM=true` is configured.

Machine cards show agent backends such as `codex`, `claude-code`, `openswarm`, and `simulated`.
